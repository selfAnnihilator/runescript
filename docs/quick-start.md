# Quick Start Guide

## Prerequisites

- **Java 21 or higher** installed on your system
- **Terminal/Command Prompt** for running commands

### Verify Java Installation
```bash
java -version
```
You should see Java 21 or higher.

## Download RuneScript

### Option 1: Direct Download
1. Go to the [GitHub repository](https://github.com/selfAnnihilator/runescript)
2. Download the JAR file from `src/runescript.jar`
3. Save it to your desired location

### Option 2: Clone Repository
```bash
git clone https://github.com/selfAnnihilator/runescript.git
cd runescript
```

## Running RuneScript Programs

### Execute a Script
```bash
java -jar runescript.jar script.rn
```

### CLI Options
```bash
# View tokenization
java -jar runescript.jar --emit-tokens script.rn

# View Abstract Syntax Tree
java -jar runescript.jar --emit-ast script.rn

# View generated bytecode
java -jar runescript.jar --emit-bytecode script.rn

# Start interactive REPL
java -jar runescript.jar
```

## Creating Your First Program

### 1. Create a new file with `.rn` extension
```bash
touch hello.rn
```

### 2. Write your first program
```runescript
print("Hello, RuneScript!");
```

### 3. Run it
```bash
java -jar runescript.jar hello.rn
```

### 4. Expected output
```
Hello, RuneScript!
Returned: null
```

## Example Programs

### Simple Math
```runescript
print(5 + 3);        // Output: 8
print(10 - 4);       // Output: 6
print(6 * 7);        // Output: 42
```

### Variables
```runescript
let name: string = "World";
let age: int = 25;
print("Hello, ", name, "!");
print("Age: ", age);
```

### Conditional Logic
```runescript
let score: int = 85;
if (score >= 90) {
    print("Grade: A");
} else if (score >= 80) {
    print("Grade: B");
} else {
    print("Grade: Needs improvement");
}
```

### Loops
```runescript
let counter: int = 0;
while (counter < 5) {
    print("Count: ", counter);
    counter = counter + 1;
}
```

## Interactive REPL

Start the interactive mode:
```bash
java -jar runescript.jar
```

Then type expressions directly:
```
> print("Hello");
Hello
Returned: null
> let x: int = 5;
> print(x);
5
Returned: null
> exit
```

## Troubleshooting

### Common Issues

**Issue: "java is not recognized"**
- Solution: Java is not installed or not in PATH
- Fix: Install Java 21+ and restart terminal

**Issue: "Could not find or load main class"**
- Solution: Not in correct directory
- Fix: Navigate to directory containing runescript.jar

**Issue: "Unable to access jarfile"**
- Solution: JAR file not in current directory
- Fix: Check current directory with `ls` (Linux/Mac) or `dir` (Windows)

### Getting Help
- Check the examples in the `examples/` directory
- Read error messages carefully (they show line numbers)
- Make sure Java version is 21 or higher
- Ensure you're in the correct directory when running commands
